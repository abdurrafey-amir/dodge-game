# Meteor Dodging Game

A simple meteor dodging game made with p5.js -> [try now](https://abdurrafey-amir.github.io/dodge-game/)

Move your player with your arrow keys and dodge as many meteors as you can! Each dodge earns you a point and each time you hit a meteor you lose a life. You initially have 5 lives (shown by the hearts on the top left of the window)<br><br>
(you need to click anywhere on the window for the sounds to start playing)
